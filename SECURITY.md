# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** disclose it in a public Issue. Contact the maintainers via:

- Create a **Private Security Advisory** in the GitHub repository, or
- Email the repository maintainers (see GitHub profile)

We will evaluate and respond as soon as possible. Thank you for responsible disclosure.
